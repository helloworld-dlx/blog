## 背景

在接触 RISC-V 和 Verilog 之前，我决定先从经典的 8051 架构入手，打下扎实的底层基础。选择 STC89C52RC 是因为它的资料非常丰富、外设生态成熟，而且可以直接通过串口下载程序，非常适合 DIY 项目。

整个设计使用嘉立创EDA（EasyEDA）完成，从原理图到PCB一手操办。

## 硬件规格

| 项目 | 参数 |
|------|------|
| 主控芯片 | STC89C52RC，LQFP44封装 |
| 电源输入 | USB-C，5V |
| 稳压芯片 | AMS1117-3.3，输出3.3V |
| 时钟电路 | 12MHz 晶振 + 2×22pF负载电容 |
| 复位电路 | 10kΩ电阻 + 10μF电容 + 手动复位按键 |
| P0口上拉 | 8×10kΩ排阻 |
| 用户外设 | 2个按键（P2.0/P2.1），2个LED（P1.0/P1.1） |
| 接口 | 全GPIO引出排针（P0/P1/P2/P3/P4）、串口编程接口（H5） |
| 电源输出 | +5V、+3.3V、GND 引出排针（H4） |
| PCB层数 | 双层板（Top/Bottom） |
| 设计版本 | V1.0（2025-09-22初版，2025-10-05更新） |

## 原理图设计

原理图按照功能模块划分成以下几个区块：

- **电源模块**：USB-C 输入经过 AMS1117-3.3 线性稳压输出 3.3V，配合多个滤波电容（C1-C4）提升供电稳定性。滑动开关控制电源通断，LED 指示灯显示电源状态。
- **主控芯片**：STC89C52RC 居中布局，去耦电容 C5 靠近 VCC 引脚放置。
- **时钟电路**：12MHz 晶振连接 XTAL1/XTAL2，负载电容 C6/C7 紧邻晶振引脚。
- **复位电路**：RC 复位网络配合手动复位按键，保证可靠启动。
- **I/O扩展**：所有 GPIO 口通过排针引出，P0 口因8051架构特性必须外加上拉电阻。
- **串口接口**：TXD/RXD 直接引出，方便串口烧录和调试。

## PCB设计

PCB 布局遵循以下原则：

- **中心化布局**：MCU 位于板卡中央，所有 GPIO 口均匀引出到四周，缩短信号走线长度。
- **双层走线**：顶层走信号线，底层大面积铺地（GND），提升抗干扰能力。
- **总线扇出**：GPIO 口到排针采用平行总线走线，整齐美观。
- **电源走线加粗**：主电源线（5V/3.3V）加宽，降低压降和阻抗。
- **丝印标注**：每个引脚、每个元件都有清晰标注，极大降低接错线的风险。
- **趣味丝印**：右下角放置了皮卡丘图案，给作品增加一点个性 😄

## 打样与焊接

板子已在外发进行打样，SMD 元件（电阻、电容、稳压芯片）自行手工焊接，直插元件（排针、晶振、按键）后焊。
